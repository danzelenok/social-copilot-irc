## Language Rule

All output must be in English only. This applies to:
- All UI text, labels, buttons, placeholders, error messages
- Code comments
- Commit messages
- Variable and function names
- Console logs

The working language between the developer and the agent may be Russian.
This does not affect the language of any code or UI output — it must always be English.
